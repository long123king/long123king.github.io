1) 获取应用程序类

 

   1: Application ExcelApp = New Application();

2) 更改 Excel 标题栏：

 

   1: ExcelApp.Caption := '应用程序调用 Microsoft Excel';

3) 添加新工作簿：

 

   1: ExcelApp.WorkBooks.Add;

4) 打开已存在的工作簿：

 

   1: ExcelApp.WorkBooks.Open( 'C:\Excel\Demo.xls' );

5) 设置第2个工作表为活动工作表：

 

   1: ExcelApp.WorkSheets[2].Activate; 
   2: ExcelApp.WorksSheets['Sheet2'].Activate;

6) 给单元格赋值：

 

   1: ExcelApp.Cells[1,4].Value := '第一行第四列';

6.5) 自动列宽

   1: excelApp.ActiveSheet.Columns[1].AutoFit();

7) 设置指定列的宽度（单位：字符个数），以第一列为例：

 

   1: ExcelApp.ActiveSheet.Columns[1].ColumnsWidth := 5;

8) 设置指定行的高度（单位：磅）（1磅＝0.035厘米），以第二行为例：

 

   1: ExcelApp.ActiveSheet.Rows[2].RowHeight := 1/0.035; // 1厘米

9) 在第8行之前插入分页符：

 

   1: ExcelApp.WorkSheets[1].Rows[8].PageBreak := 1;

10) 在第8列之前删除分页符：

 

   1: ExcelApp.ActiveSheet.Columns[4].PageBreak := 0;

11) 指定边框线宽度：

 

   1: ExcelApp.ActiveSheet.Range[ 'B3:D4' ].Borders[2].Weight := 3;
1-左 2-右 3-顶 4-底 5-斜( \ ) 6-斜( / )

12) 清除第一行第四列单元格公式：

 

   1: ExcelApp.ActiveSheet.Cells[1,4].ClearContents;

13) 设置第一行字体属性：

 

   1: ExcelApp.ActiveSheet.Rows[1].Font.Name := '隶书';
   2: ExcelApp.ActiveSheet.Rows[1].Font.Color := clBlue;
   3: ExcelApp.ActiveSheet.Rows[1].Font.Bold := True;
   4: ExcelApp.ActiveSheet.Rows[1].Font.UnderLine := True;

14) 进行页面设置：


a.页眉：

   1: ExcelApp.ActiveSheet.PageSetup.CenterHeader := '报表演示';
b.页脚：
   1: ExcelApp.ActiveSheet.PageSetup.CenterFooter := '第&P页';
c.页眉到顶端边距2cm：
   1: ExcelApp.ActiveSheet.PageSetup.HeaderMargin := 2/0.035;
d.页脚到底端边距3cm：
   1: ExcelApp.ActiveSheet.PageSetup.HeaderMargin := 3/0.035;
e.顶边距2cm：
   1: ExcelApp.ActiveSheet.PageSetup.TopMargin := 2/0.035;
f.底边距2cm：
   1: ExcelApp.ActiveSheet.PageSetup.BottomMargin := 2/0.035;
g.左边距2cm：
   1: ExcelApp.ActiveSheet.PageSetup.LeftMargin := 2/0.035;
h.右边距2cm：
   1: ExcelApp.ActiveSheet.PageSetup.RightMargin := 2/0.035;
i.页面水平居中：
   1: ExcelApp.ActiveSheet.PageSetup.CenterHorizontally := 2/0.035;
j.页面垂直居中：
   1: ExcelApp.ActiveSheet.PageSetup.CenterVertically := 2/0.035;
k.打印单元格网线：
   1: ExcelApp.ActiveSheet.PageSetup.PrintGridLines := True;

15) 拷贝操作：


a.拷贝整个工作表：

   1: ExcelApp.ActiveSheet.Used.Range.Copy;
b.拷贝指定区域：
   1: ExcelApp.ActiveSheet.Range[ 'A1:E2' ].Copy;
c.从A1位置开始粘贴：
   1: ExcelApp.ActiveSheet.Range.[ 'A1' ].PasteSpecial;
d.从文件尾部开始粘贴：
   1: ExcelApp.ActiveSheet.Range.PasteSpecial;

16) 插入一行或一列：

 

   1: ExcelApp.ActiveSheet.Rows[2].Insert;
   2: ExcelApp.ActiveSheet.Columns[1].Insert;

17) 删除一行或一列：

 

   1: ExcelApp.ActiveSheet.Rows[2].Delete;
   2: ExcelApp.ActiveSheet.Columns[1].Delete;

18) 打印预览工作表：

 

   1: ExcelApp.ActiveSheet.PrintPreview;

19) 打印输出工作表：

 

   1: ExcelApp.ActiveSheet.PrintOut;

20) 工作表保存：

 

   1: if (! ExcelApp.ActiveWorkBook.Saved )
   2: {
   3:     ExcelApp.ActiveSheet.PrintPreview;
   4: }

21) 工作表另存为：

 

   1: ExcelApp.SaveAs( 'C:\Excel\Demo1.xls' );

22) 放弃存盘：

 

   1: ExcelApp.ActiveWorkBook.Saved := True;

23) 关闭工作簿：

 

   1: ExcelApp.WorkBooks.Close;

24) 退出 Excel：

 

   1: ExcelApp.Quit;
